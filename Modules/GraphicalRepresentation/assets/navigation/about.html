INTRODUCTION

  • Twitter tweets contain various types of information, which include health-related information. Analysis of health-related tweets would help us understand health conditions and concerns encountered in our daily life. In this work, we evaluated an approach to extracting causal relations from tweets using natural language processing (NLP) techniques.
  • With many organization using twitter to post updates and news on the topics that they cover. As a result, the huge number of producers of information and updates by each production team from across the globe generate huge data for a specific topic.
  • By taking in the huge datasets we aim to provide as accurate and aggregated data to the users using the Twitter API.
  • In this global pandemic, the importance of information to the research space and the local and global coordinated effort to tackle this and future emergency situations is vital. As a result , we are motivated to bring a system to be built upon to provide real-time access to large data while filtering to their needs.

LDA

  • LDA is an unsupervised machine learning algorithm that allows a a set of textual observations to be explained by unobserved groups that explain similarities within the data. LDA represents documents as mixtures of topics that spit out words with certain probabilities.
  • Latent Dirichlet allocation (LDA) is a generative probabilistic model of a corpus. The basic idea is that documents are represented as random mixtures over latent topics, where each topic is characterized by a distribution over words.
